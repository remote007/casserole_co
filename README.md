# Project Title : 
Casserole Co

## Project Description : 
A platform to offer you platefull of fresh and delicious dishes at your doorstep.

## Tech Stacks : 
HTML
CSS
JavaScript

## Features : 

#### Home Page
Customer lands on homepage of the website and views dishes available .

#### Sign up and Login
User is asked to login or signup depending upon their requirement of placing an order. They get the authority to place an order and modify the cart after login.

### How to Install and Run the Project : 
1. clone the project to your local system.
2. npm init.
3. go live via vscode or open index.html on browser.
